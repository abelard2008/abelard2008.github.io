Hash

Hash is a technique which addresses records provided with an identifier called primary key, or simply key.

If two conditions hold, a hash table supports constant expected-time element access: the elements must be relatively evenly distributed among the slots, and the number of slots must vary in direct proportion to the number of elements.

Given a good hashing function, presering constant-time access to elements requires keeping hash chains short. Keeping chains short without wasting space requires adjusting the number of slots as the number of elements changes.1

two types of popular hash functions

  • division method

h(k) = k mod M, where M is prime

  • multiplication method

h(k) = ⌊ m (k A mod 1) ⌋ 1, where A is in the range 0 < A < 1, kA mod 1 is the fraction of the kA, in general, A is 0.6180339887 2

An advantage of the multiplication method is that the value of m is not critical. We typically choose it to be a power of 2 (m = 2p for some integer p), since we can then easily implement the function on most computers as follows. Suppose that the word size of the machine is w bits and that k fits into a single word. We restrict A to be a fraction of the form s=2w, where s is an integer in the range 0 < s < 2w. we first multiply k by the w-bit integer s = A * 2w. The result is a 2w-bit value r1 * 2w + r0, where r1 is the high-order word of the product and r0 is the low-order word of the product. The desired p-bit hash value consists of the p most significant bits of r0. 1

why r1 * 2w + r0 is equal to ⌊ M (kA mod 1) ⌋, the reasoning steps is as follows:

⌊ m ( kA mod 1) ⌋ ⇔ ⌊ m \(\frac{k * s}{2^{w}}\) mod 1 ] ⌋ ⇔

⌊ m (\(\frac{k * s}{2^{w}}\) mod 1) ⌋ ⇔ ⌊ m (\(\frac{ r_{1} * 2^{w}+ r_{0}}{2^{w}}\) mod 1) ⌋ , where k * s can be equal to r1 * 2w + r0, and

\(\frac{r_{0} * 2^{w} + r_{1}}{2^{w}}\) mod 1 ⇔ r1 + \(\frac{r_{0}}{2^{w}} \)

=> \(\frac{r_{1}}{2^{w}}\) ⇔ fraction of kA

=> h(k) = ⌊ m \(\frac{r_{0}}{2^{w}}\) ⌋ = ⌊ 2p \(\frac{r_{0}}{2^{w}}\) ⌋

and because X * 2p is X left shift p-bits and X / 2p is right shift p-bits, so that ⌊ 2p \(\frac{r_{0}}{2^{w}}\) ⌋ ⇔ ⌊ \( \frac{r_{0}}{2^{w-p} \) ⌋, which is said that h(k) is p most significant bits of r0.
